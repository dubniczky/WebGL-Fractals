## Usage

### 1. Download repository

```bash
https://github.com/dubniczky/WebGL-Fractals
cd WebGL-Fractals
```

### 2. Install dependencies

```bash
make install
```

or

```bash
yarn install
```

### 3. Build with webpack

```bash
make build
```

or

```bash
yarn build
```
